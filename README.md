## What Is This

A conversational portfolio where visitors chat with an AI that knows exactly one person — me — and refuses to discuss anything else. Think ChatGPT but with crippling scope limitations and zero existential dread.

The landing page greets you, shows a random tech quote from someone smarter than me, and presents a chat box. Ask it about my skills, projects, experience, or philosophy. It answers from a strict context file. Ask it about the weather and it politely tells you to go away.

When the AI backend is down (it happens, I'm not made of money), it cracks a joke about tripping over loose tokens and shows you clickable section shortcuts instead. Because even my error states have better UX than most people's landing pages.

## Running This Locally

```bash
git clone https://github.com/shivang4857/portfolio.git
cd portfolio
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Marvel at the typography. Ask the AI something. Judge my career choices.

## Using This as a Template

You want to steal this for your own portfolio? Respectable. Skip the steps — run the site, open the chat, and ask the AI how to fork it. It has a specialization in helping people copy other people's work. It will give you a better response than any README bullet points. I built a guide that talks back. The irony is not lost on me.
